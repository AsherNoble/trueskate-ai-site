// THE HAND

The hand.

Almost every robotic hand is built to grip. TrueSkate-AI's needs the opposite — speed and human-like motion under no load — so it borrows from several open-source designs and recombines them.

The base is the ORCA Hand (ETH Zurich): five-finger, anthropomorphic, and tendon-driven from the forearm, which keeps the fingers light enough to move fast. It has per-finger splay for spread-and-pinch gestures, and it's built around the same serial-bus servos already on the bench. Its grasping-focused "Lite" variant was ruled out — it gives up the independent per-finger splay this project depends on. Where their finger mechanisms suit a speed-over-strength brief better, parts are borrowed from Pollen Robotics' Amazing Hand and Ruka-v2.

Each finger gets a tendon-driven curl plus base splay — roughly two to three degrees of freedom — so it can both flick and spread. The fingertips make capacitive contact with the screen, with touch-down and lift gated electronically so every tap registers cleanly: a reliability layer under real lift-and-press motion, not a substitute for it. The hand actually moves like a hand.

The first build is a two-finger scaffold — index and middle — to prove the curl-and-splay module and the control loop before scaling to all five.

  • ~2–3 DOF per finger
  • tendon-driven
  • zero load-bearing
  • forked from open-source hands
