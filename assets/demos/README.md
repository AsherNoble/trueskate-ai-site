# Demo clips

Drop the looping clips here. Until a file is present, the page renders a
styled "clip pending" placeholder in its slot — the layout never breaks.

Render each as: `<video muted loop playsinline autoplay preload="metadata">`.

Expected files (referenced by the page):

| File          | Used in            | Shows                                   |
|---------------|--------------------|-----------------------------------------|
| `device.mp4`  | hero `FIG.01`      | phone screen mid-trick                  |
| `ocr.mp4`     | hero `FIG.02`      | detected trick text highlighted         |
| `gesture.mp4` | hero `FIG.03`      | swipe path drawn over the screen        |
| `landed.mp4`  | hero `FIG.04`      | a clean landed trick                    |
| `hero-extra.mp4` | hero (optional) | another trick clip                      |
| `trick-01.mp4` … `trick-06.mp4` | demo gallery | pure trick showcase (unedited rollouts) |

Keep them short, muted, and small (≈540p, a few seconds, looped).
